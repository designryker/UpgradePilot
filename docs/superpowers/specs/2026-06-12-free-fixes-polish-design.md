# Free Performance Fixes Polish Design

## Goal

Make Free Performance Fixes a concise, trustworthy action checklist rather than
a general Windows optimization article.

## Content Rules

- Each item is one direct action sentence.
- Keep actions that are broadly useful or directly triggered by the diagnosis.
- Do not impose an arbitrary five-item limit.
- Do not include generic debloat advice, visual-effects tweaks, or temporary-file
  cleanup as performance fixes.
- Avoid claims that High Performance always improves FPS.

## Core Actions

- Disable unnecessary startup apps.
- Desktop: use Balanced and make sure Power Saver is off.
- Laptop: plug in the charger and enable Windows/OEM performance mode.
- Set the monitor to its maximum refresh rate.
- Close unused browsers, launchers, recording tools, and overlays before gaming.

## Conditional Actions

- Update GPU driver when GPU is relevant.
- Update chipset driver when CPU is relevant.
- Enable and verify XMP/EXPO when memory speed is relevant.
- Check CPU/GPU temperatures when cooling or the diagnosed component is relevant.
- Move the game to SSD/NVMe when the selected game drive is HDD.
- Keep diagnostic validation actions where they directly confirm the bottleneck.

## Verification

- Smoke checks require the concise core actions.
- Smoke checks reject removed low-value wording.
- Existing build, brand, and part consistency checks pass.
